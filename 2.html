# 2. Erste Schritte

Ziel ist es, ein "Raumschiff-Spiel" entwickeln, das an den Klassiker Asteroids angelegt ist, aber für 2 Personen spielbar ist, die um einen höheren Punktestand spielen.

Aufgabe 2.1

Lade die erste (Grund-)Version des Spiels herunter und führe sie aus: asteroids-1.zip

Schau dir an, wie die Klasse Player definiert ist: Wichtig ist das Schlüsselwort class. Die Methode init() ist der Konstruktur, der aufgerufen wird, wenn ein neues Objekt der Klasse erzeugt wird.

class Player():

    def __init__(self, name):
        self.name = name
        self.lifes = 2
        self.x = 0
        self.y = 20

    def draw(self):
        screen.draw.text(self.name,(self.x,self.y))
        x = self.x
        i = 0
        while i < self.lifes:
            screen.blit("heart",(x,self.y+10))
            x += 50
            i += 1

Bitte merken

self steht immer für das aktuelle Objekt. Es ist immer der erste Parameter für eine Methode eines Objekts und muss immer angegeben werden, wenn auf eine Eigenschaft des Objekts zugegriffen wird.

Dem Konstruktor können Parameter übergeben werden, damit Eigenschaften des Objekts, wie hier der Name des Spielers direkt festgelegt werden.

player1 = Player("Alice")

Bei der Definition der Klasse Ship steht Actor, damit ist eine Spezialisierung der Klasse Actor angegeben, d.h. ein Ship erbt alle Eigenschaften und Methoden eines Actors (kann sich z.B. zeichnen). Die Objekte der Klasse Ship verfügen zusätzlich über einige besondere Eigenschaften und Methoden, z.B. um die Bewegung des Schiffs umzusetzen. So kann bei einem Ship für einige Sekunden ein Turbo eingeschaltet werden, welcher die Geschwindigkeit erhöht.

Es soll nun ein zweiter Spieler hinzugefügt werden. Da alles bereits objektorientiert programmiert wurde müssen dazu lediglich neue Objekte erzeugt (und die Tastatursteuerung für das zweite Ship ergänzt) werden.

Aufgabe 2.4

  • Füge einen zweiten Spieler hinzu. So dass sein/ihr Name, Anzahl Leben und ein Raumschiff angezeigt werden. (Verwende als X-Koordinate 1000 und achte darauf nicht nur die Objekte zu erzeugen, sondern in der Funktion draw() ihnen auch den Aufruf zu geben sich zu zeichnen.
  • Ändere die Anzahl der Leben aller Spieler auf 3, durch eine Änderung im Konstruktor.

Es sollen nun 10 Asteroiden (Bild: "asteroid") hinzugefügt werden, die an zufälligen Positionen mit zufälligen Richtungen erscheinen und sich in diese Richtung weiterbewegen. Die Berechnung der neuen X- und Y- Position kannst du von der Klasse Ship abschauen.

Aufgabe 2.5

  • Erstelle eine neue Klasse Asteroid und mit Hilfe einer Zählschleife 10 Objekte davon. Achtung: Damit die Objekte auch gezeichnet werden, musst du ihre Methode draw() auch aufrufen. Schau dir dazu an, wo die Methode draw() der Raumschiffe aufgerufen wird und nutze eine Schleife um allen Asteroiden Anweisungen zu geben!
Lösung 
class Asteroid(Actor):

    def __init__(self, x, y):
        super().__init__("asteroid")
        self.x = x
        self.y = y
        self.angle = random.randint(0, 360)
        self.speed = random.randint(50, 100)

# Asteroiden erzeugen
asteroids = []
for i in range(10):
  a = Asteroid(random.randint(0, WIDTH), random.randint(0, HEIGHT))
  asteroids.append(a)
