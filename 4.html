    # 4. Energielevel

    Um das Raumschiff zu "befragen" wieviel Energie es noch hat wird die Methode get_energy() benutzt. Das Schlüsselwort return sorgt dafür, dass derjenige, der die Methode aufruft einen Wert zurück erhält.

    def get_energy(self):
    return self.max_energy - len(self.shots)
    1
    2

    Um die Methode zu testen, kannst du nach dem Erstellen eines Raumschiffs eine Zeile wie die folgende benutzen.

    # Die Energie wird als Zahl zurückgegeben und muss vor der Ausgabe erst mit *str()* in eine Zeichenkette umgewandelt werden
print("Energie von Ship1: " + str(ship1.get_energy()) )

    2

    Aufgabe 4.1

    • Verändere die Methode wie oben angegeben.
    • Überprüfe, wie hoch das Energielevel eines neuen Raumschiffs ist
    • Schreibe einen Kommentar über die Methode, die erklärt, wovon die Höhe der Energie abhängt

    Damit Raumschiffe das Player-Objekt "kennen", das dafür zuständig ist, die Anzal der Leben zu verwalten und die Daten anzuzeigen besteht bereits eine Assoziation zwischen den Klassen Ship und Player. Nach dem Erstellen der Ship und layer Objekte geschieht dies durch die folgende Zeile:

    ship1.player = player1
    1

    Damit das Player Objekt auch das jeweilige Raumschiff kennt und damit das Raumschiff nach seinem Energiestand gefragt werden kann, ist es nötig, diese Assoziation auch von der Klassse Ship zur Klasse *Player aufzubauen:

    ship1.player = player1
player1.ship = ship1
