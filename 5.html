# 5. Schüsse

Ein Raumschiff soll die Asteroiden zerschiessen können und dadurch Punkte erhalten. Dazu wird die Klasse Shot benutzt: Sie soll eine Spezialisierung von Actor sein und im Konstruktor die Attribute center und angle setzen. Die Geschwindigkeit eines Shots soll 10 betragen und außerdem soll das Schiff, zu dem dieser Schuss gehört gespeichert werden (Assoziation).

class Shot(Actor):

    def __init__(self, center, angle, ship):
        super().__init__("laser_shot")
        self.pos = center
        self.angle = angle
        self.speed = 10
        self.ship = ship

    def update(self):
        self.x = self.x + (self.speed*math.cos(math.radians(self.angle)))
        self.y = self.y - (self.speed*math.sin(math.radians(self.angle)))

        if self.x < 0 or self.x > WIDTH or self.y < 0 or self.y > HEIGHT:
            self.ship.shots.remove(self)
        else:
            for a in asteroids:
                if self.colliderect(a):
                    # a.explode() # Später diesen Kommentar entfernen
                    # Hier 100 zu self.ship.player.points addieren
                    if self in self.ship.shots:
                        self.ship.shots.remove(self)

Aufgabe 5.1

  • Erstelle die Klasse Shot.

Ein Raumschiff muss die von ihm abgegebenen Schüsse "kennen" (Assoziation), damit die Schüsse weiter bewegt werden und in der Berechnung der verfügbaren Energie berücksichtigt werden können.

Aufgabe 5.2

  • Verändere die Methode shoot() der Klasse Ship, so dass ein Schuss abgegeben wird, falls die Anzahl der Schüsse kleiner ist als die maximale Energie.
Lösung 
def shoot(self):
    if (len(self.shots) < self.max_energy):
        shot = Shot(self.center, self.angle, self)
        self.shots.append(shot)

Aufgabe 5.4

  • Suche die Stelle im Programmcode, die dafür sorgt, dass die Methode update() der Klasse Shot aufgerufen wird.
Tipp

Sie befindet sich in der Klasse Ship

Falls ein Asteroid von einem Schuss getroffen wird, so sollen an seiner Stelle zwei kleinere Asteroiden erscheinen (Bild: "asteroid_small"). Dies soll allerdings nur geschehen, wenn der getroffene Asteroid nicht selber schon das Bild "asteroid_small" benutzt.

Aufgabe 5.5

  • Erstelle die Methode explode() für einen Asteroiden und rufe die Methode auf, indem du die entsprechende Stelle in der Methode update() der Klasse Asteroid einkommentierst (also das Doppelkreuz entfernst).
Lösung 
def explode(self):
    asteroids.remove(self)
    if self.image is not "asteroid_small":
        for i in range(2):
            a = Asteroid(self.x, self.y)
            a.image = "asteroid_small"
            asteroids.append(a)
